a. ¿Quién inventó el sistema de control de versión Git y por qué?

Git fue diseñado por Linus Torvalds pensando en un sistema de control de versiones con los atributos de velocidad, eficiencia, diseño sencillo, soporte para desarrollo no lineal (ramas paralelas), distribuido y capaz de manejar grandes proyectos

b. ¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dólares. Microsoft compro Github para acercarse más al software libre y como estrategia para tener un contacto más directo con los desarrolladores.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Si, se pueden utilizar diferentes herramientas que facilitan el trabajo con git de forma gráfica y sencilla
